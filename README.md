# 🚀 Project & Task Management System

## 📌 Overview

The **Project & Task Management System** is a full-stack web and mobile application developed as part of an internship task at **Debo Engineering**.

This system helps organizations efficiently manage:

* Projects
* Teams
* Tasks
* Progress tracking

It replaces manual tools like Excel sheets and messaging apps with a centralized digital platform.

---

## 🎯 Objectives

* Manage projects and teams efficiently
* Assign and track tasks
* Monitor progress and deadlines
* Provide real-time updates for team members
* Support both **web and mobile platforms**

---

## 🏗️ Tech Stack

### 💻 Web Development

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MySQL

### 📱 Mobile Development

* React Native
